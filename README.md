# Drum Kit 🥁

A web-based interactive drum kit built with HTML, CSS, and JavaScript. Press the buttons or corresponding keyboard keys to play different drum sounds.

## 🚀 How It Works
- Each button corresponds to a drum sound (`w`, `a`, `s`, `d`, `j`, `k`, `l`)
- Pressing a button or keyboard key triggers the respective drum sound
- Buttons animate briefly when pressed for visual feedback

## 🛠 Built With
- HTML
- CSS
- JavaScript (Event Listeners & Audio API)
- Google Fonts: Arvo

## 📚 What I Learned
- addEventListener() for click and keypress events
- Playing audio with JavaScript (`new Audio().play()`)
- Adding/removing CSS classes dynamically for animations
- Structuring interactive web apps

## 💻 How to Run
1. **Clone the repository**  
```bash
git clone https://github.com/your-username/drum-kit-js.git
```
2. **Open the project folder**
3. **Double-click index.html**
- The drum kit will open in your default browser
4. **Or open via a code editor live server**
- If you use VS Code, install the Live Server extension
- Right-click index.html → Open with Live Server
⚡ No backend needed—pure front-end project
